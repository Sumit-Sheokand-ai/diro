# Changelog

All notable changes to DIRO will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-17

### Added
- Initial release of DIRO
- Network Scanner with host discovery and DNS lookup
- Multi-threaded Port Scanner with service detection
- Subdomain Enumeration module
- Hash Analyzer (identification and generation)
- Web Header Analysis with security header checking
- DNS Lookup tools (forward and reverse)
- Ping Sweep functionality
- WHOIS and SSL certificate information helpers
- Colorful terminal interface
- Legal disclaimer and user agreement
- Installation script for Kali Linux
- Comprehensive documentation

### Features
- 10 different security testing modules
- Menu-driven interface
- Cross-platform support (Linux/Windows)
- Multi-threaded operations for performance
- Common port scanning presets
- MD5, SHA1, SHA256, SHA512 hash support
- HTTP security header validation

### Documentation
- README.md with installation and usage instructions
- LICENSE file (MIT)
- CONTRIBUTING.md guidelines
- Code comments and docstrings
